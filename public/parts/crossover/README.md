# Изображения деталей кузова (Кроссовер)

Положите сюда PNG/JPG файлы деталей. Имена файлов должны совпадать с путями из `src/data/partsCatalog.js`:

| Файл | Деталь |
|------|--------|
| hood.png | Капот |
| door_front.png | Передняя дверь |
| door_rear.png | Задняя дверь |
| fender_front.png | Переднее крыло |
| fender_rear.png | Заднее крыло |
| trunk.png | Багажник |
| roof.png | Крыша |

Если файла нет — в редакторе показывается серый placeholder с названием детали.
